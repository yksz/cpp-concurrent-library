# What is this?
This is a header only concurrent library for C++.

This library's main features:
- Actor
- Blocking queue
- Countdown latch
- Publish-subscribe
- Scheduler
- Thread pool

This library uses **"any"** of boost library.

# Installation
```shell
./build.sh
cd build
make
sudo make install
```
or

Copy files in src directory to your project

# System Requirements
C++11 or later
